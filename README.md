# Chunkbase Seed Map Coordinate Convertor

Convert `tp commad` to xyz coordinate

`/tp -16024 ~ 6664` to `-16024 ~ 6664` and nether coordinte (divide by 8) `-2003 ~ 833`

> ~ at y axis mean that we do care about the value of height in minecraft
